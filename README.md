# KeyXad - Portal de Consorcios

Sistema integral de gestión para propietarios y administraciones de consorcios. Desarrollado con **Laravel 12** y **Tailwind CSS**.

## - Características
- **Acceso mediante QR:** Autenticación automática para vecinos.
- **Cuenta Corriente:** Visualización detallada de Debe/Haber con saldos acumulados.
- **Gestión de Expensas:** Descarga de comprobantes en PDF.
- **Dashboard Moderno:** Interfaz minimalista con Glassmorphism.

##  Tecnologías utilizadas
- **Backend:** PHP 8.2+ / Laravel 12
- **Frontend:** Tailwind CSS / Blade / Bootstrap Icons
- **Base de Datos:** MySQL
- **Herramientas:** Composer, Git, WSL2

##  Instalación Local
Si queres probar este proyecto en tu máquina:

1. Clona el repositorio:
   `git clone https://github.com/Elio1204/tu-repositorio.git`
2. Instala las dependencias:
   `composer install`
3. Configura tu archivo `.env` con tus credenciales de base de datos.
4. Genera la clave de la aplicación:
   `php artisan key:generate`
5. Ejecuta el servidor:
   `php artisan serve`
